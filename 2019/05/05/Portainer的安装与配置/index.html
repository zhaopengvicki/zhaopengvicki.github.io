Portainer的安装与配置

发表于 2019-05-05

目前对于Docker的图形化管理工具主要分为三种：Docker UI、Portainer、Shipyard。

三种容器管理工具有着各自的优缺点，笔者结合自身喜好，最终选择了Portainer：

  1. Portainer相比较Docker UI而言，有着容器内的日志展示，并且可以在页面上通过命令行的方式直接向容器内部发送shell命令。
  2. Portainer相比较Shipyard，在安装时与部署时更为简单，只需要一个容器足已满足服务需要，而Shipyard在安装时却需要众多容器相互协作才可提供服务。

服务环境

  • 操作系统：CentOS 7.4 64位
  • Docker版本：1.13.1 (yum安装的默认版本）
